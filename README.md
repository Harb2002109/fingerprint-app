# تطبيق المصادقة بالبصمة - Kivy

تطبيق أندرويد بلغة Python يستخدم Kivy لمصادقة البصمة الإصبعية.

## الملفات
- `main.py` - الكود الرئيسي للتطبيق
- `buildozer.spec` - إعدادات البناء
- `.github/workflows/build.yml` - إعداد GitHub Actions

## كيفية البناء
1. ارفع المشروع إلى GitHub
2. اذهب إلى Actions → Build Kivy APK
3. انتظر 15-30 دقيقة
4. حمّل APK من Artifacts

## الميزات
- تسجيل الدخول بالبصمة أو كلمة المرور
- إنشاء حساب جديد مع تسجيل بصمة
- حفظ بيانات المستخدم
